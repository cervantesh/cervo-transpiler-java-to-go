# Code of Conduct

This project expects direct, respectful, technical collaboration.

## Expected Behavior

- Be professional and constructive.
- Focus criticism on code, design, tests, documentation, and reproducible evidence.
- Assume public project spaces may be read by future users and contributors.
- Respect privacy and confidentiality boundaries.
- Give clear attribution when using external material.

## Unacceptable Behavior

- Harassment, threats, insults, or personal attacks.
- Publishing private information without permission.
- Submitting proprietary, confidential, or stolen code.
- Posting secrets, tokens, credentials, or customer data.
- Misrepresenting AI-generated content as deterministic project output.

## Enforcement

Maintainers may edit, hide, or remove comments, issues, pull requests, or other contributions that violate this code of conduct. Serious or repeated violations may result in blocked participation.

If a conduct issue also includes a security or privacy concern, follow `SECURITY.md`.
