# Corpus Evidence

This directory is produced by:

```bash
go run ./tools/corpus
```

The evidence is intentionally separated from `.corpus/`:

- `.corpus/` contains local third-party repository clones and is ignored by git.
- `evidence/corpus/` contains deterministic scan, report, and migration artifacts that can be committed for review.

Each repository directory should contain:

- `source.json`: pinned source metadata.
- `scan.txt`: project scan output.
- `report.json`: deterministic machine-readable migration report.
- `report.md`: human-readable migration report.
- `migration.md`: migration report generated by `j2go migrate`.
- `migration.log`: deterministic operation log.
- `migrate.txt`: migration command stdout/stderr transcript.
- `go-test.txt`: generated Go verification transcript.
- `go/`: generated Go output when deterministic migration can emit files.
